# Oliv Studio

A modern responsive branding agency landing page built with pure **HTML & CSS**.
This project focuses on recreating a clean and premium agency-style UI with responsive layouts and smooth visual design.

## About

The goal of this project was to recreate a modern branding agency website design as close as possible using only HTML & CSS.

This project helped in practicing:

* Responsive Web Design
* CSS Flexbox
* CSS Grid
* Layout Structuring
* Modern UI Styling
* Positioning & Spacing
